Student Performance SQL Analysis

Project Overview
This project analyzes student performance data using SQL to understand the impact of various factors such as study time, alcohol consumption, family support, and absences on academic results.

Tools & Skills Used
* SQL (MySQL)
* Joins & UNION
* Aggregations
* Data Analysis
* Business Insights

Dataset Information
The dataset contains student data from two schools:
GP (Gabriel Pereira)
MS (Mousinho da Silveira)

It includes:
 Academic scores (G1, G2, G3)
 Study habits
 Lifestyle factors
 Family background

Key Analysis Performed

Academic Analysis
* Comparison of grades across schools
* Study time vs performance
* Failure impact on grades

Lifestyle Analysis
* Alcohol consumption vs grades
* Internet access impact
* Social activity impact

Behavioral Insights
* Absences vs performance
* Family support influence

Key Insights
- Higher study time leads to better final grades
- Students with more failures tend to have lower performance
- Alcohol consumption negatively impacts grades
- Students with internet access perform slightly better
- Absences are negatively correlated with performance
